Angular Material is a UI component library.
Provides us with component to build awesome user interfaces in quick time.
Implementation of Google's material design specification

for example 
React   --> Material UI
Angular --> Angular Material
Vue     --> Vuetify

HTML,
CSS,
JAVASCRIPT,
